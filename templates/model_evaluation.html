Performance Analysis and Interpretation

The models are compared using domain-appropriate metrics for Colombian public library analytics:

  • Accuracy: Percentage of libraries where internet connectivity is correctly predicted. Important for overall resource planning accuracy.
  • Precision: Of libraries predicted to have internet, what percentage actually do. High precision avoids false positive assumptions about connectivity.
  • Recall: Of libraries that actually have internet, what percentage are correctly identified. High recall ensures no connected libraries are overlooked.
  • F1-score: Balanced metric combining precision and recall, ideal when both false positives and false negatives have costs.
  • MAE (Mean Absolute Error): Average prediction error in user count. Lower MAE indicates more reliable user engagement forecasts for resource allocation.
  • R² (Coefficient of Determination): Proportion of variation in user counts explained by the model. Higher R² suggests the model captures true library engagement patterns.
