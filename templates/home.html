Project Overview

This academic project documents the initial stages of a machine learning initiative for Colombian public libraries. The application is designed to show:

  • Business understanding of a real world problem.
  • Data understanding of public library datasets from Colombian open data portals.
  • Data engineering steps that prepare the data for future modeling.

Goals for the Flask Application

  • Build a navigation menu with the required CRISP-ML phases.
  • Document the dataset, data quality, and cleaning process in English.
  • Use the existing dataset definitions located in pipeline.py.
  • Prepare the project for deployment to Render.com.
